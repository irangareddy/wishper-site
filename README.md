# wishper-site

Landing page for [Wishper](https://github.com/irangareddy/wishper-app) — live at <https://wishper.irangareddy.in>.

Single static `index.html`, no build step. Served via GitHub Pages from `main`.

## Deploy

GitHub Pages auto-deploys every push to `main`. ~30 seconds from commit to live.

## DNS

CNAME record at `wishper.irangareddy.in` → `irangareddy.github.io`.
